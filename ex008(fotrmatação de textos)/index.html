Principais formatações

Negrito / Destaque

Frase com termo em negrito com a tag B (forma)

Frase com termo em destaque usando a tag strong (semântica)

Itálico / ênfase

Esta frase tem um termo em italico, usando a tag I (não semântica)

Esta frase tem um item em ênfase usando a tag EM (semântica)

Texto marcado

Podemos marcar um texto com a tag MARK.

Vamos marcar o texto novamente.

Texto grande e pequeno

O texto grande usa a tag BIG(obsoleta) e o texto pequeno se usa a tag SMALL.

Texto excluído.

O texto excluído deve ser lido, mas não deve ser considerado.

Texto inserido

O texto inserido serve pra dizer que ele foi adicionado depois.

Texto sobrescrito

Serve para adicionar coisas do tipo: x10 + 4

Texto subscrito

Serve para inserir coisa como: H2O
